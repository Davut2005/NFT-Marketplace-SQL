# E-Commerce Online Shop

This project is developed to practice **SQL database design and data analysis concepts** using **PostgreSQL**, as part of a university Database course.

The project models a real-world **E-Commerce system**, following proper **ER modeling**, **conceptual design**, and **relational implementation** principles.

---

## Project Goals
- Design a relational database based on a real-world E-Commerce scenario
- Apply ER and conceptual modeling concepts
- Implement at least 7 related entities (tables)
- Practice advanced SQL features:
  - JOINs across multiple tables
  - GROUP BY and HAVING clauses
  - Views (simple and complex)
  - Referential integrity and cascading deletes

---

## Database Features
- Independent, dependent, and associative tables
- Primary keys and foreign keys
- ON DELETE CASCADE constraints
- Many-to-many relationships
- Sample data for testing queries
- SQL views for reporting and analysis
